# Mon Budget

Application web de suivi de budget personnel, en temps réel.

➡️ **Utiliser l'application : https://hounkponou.github.io/mon-budget/**

## Fonctionnement
- 100 % dans le navigateur, aucune donnée personnelle dans ce dépôt.
- Les données sont chargées et sauvegardées automatiquement depuis un **dépôt GitHub privé** séparé
  (`mon-budget-data`), via un jeton d'accès personnel que tu saisis dans l'onglet **Réglages**.
- Fonctionne aussi hors-ligne (cache navigateur) une fois chargée.

## Confidentialité
Ce dépôt est public mais ne contient **que du code** — aucun montant, aucune transaction.
Tes chiffres restent dans ton dépôt privé.
